# TaskFlow Pro

TaskFlow Pro is a responsive task management web application built using HTML, CSS, and JavaScript.

## Features

* User Registration and Login
* Form Validation
* Landing Page
* Task Management (CRUD Operations)
* Search Tasks
* Filter Tasks by Status
* Local Storage Data Persistence
* Responsive Design for Mobile and Desktop
* Toast Notifications
* Smooth Hover Effects and Transitions

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Local Storage

## Project Structure

TaskFlow-Pro/

* signup.html
* login.html
* dashboard.html
* style.css
* auth.js
* dashboard.js
* README.md

## How to Run

1. Open the project in VS Code.
2. Open login.html using Live Server.
3. Create a new account.
4. Login using your credentials.
5. Manage tasks using the dashboard.
