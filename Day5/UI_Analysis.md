# Activity 1 – UI Analysis

# 1. Spotify

## Navigation

Spotify provides a simple and intuitive navigation system. The bottom navigation bar contains options such as Home, Search, and Library, allowing users to move easily between different sections of the application. The navigation structure is consistent throughout the platform, making it easy for users to find content.

## Layout

Spotify uses a clean card-based layout to display playlists, albums, artists, and recommendations. The content is organized in a structured manner, which helps users browse music efficiently. The layout maintains consistency across different screens.

## Typography

Spotify uses modern and readable fonts. Different font sizes and weights are used to create a clear visual hierarchy. Important information such as playlist names and artist names is highlighted effectively.

## Colors

The application primarily uses a dark theme with green accent colors. This color combination creates a modern and premium appearance while reducing eye strain during extended usage.

## User Experience

Spotify offers an excellent user experience through personalized recommendations, easy navigation, and fast performance. Features such as playlists, music suggestions, and search functionality enhance user satisfaction and engagement.

# 2. Google Maps

## Navigation

Google Maps provides easy-to-access navigation controls. The search bar is prominently placed, and important features such as directions, saved places, and location sharing are easily accessible.

## Layout

The application dedicates most of the screen space to the map while keeping essential controls visible. This layout ensures that users can focus on navigation without unnecessary distractions.

## Typography

Google Maps uses clear and readable typography for road names, locations, and navigation instructions. The text is designed to remain visible and understandable even while users are moving.

## Colors

The application uses different colors to represent roads, traffic conditions, water bodies, parks, and landmarks. These colors help users quickly understand map information and improve navigation efficiency.

## User Experience

Google Maps provides a highly effective user experience through accurate directions, real-time traffic updates, and location-based recommendations. The application minimizes user effort and helps users reach destinations efficiently.

# 3. Canva

## Navigation

Canva offers a well-structured navigation system with clearly labeled sections such as Templates, Projects, Brand Kit, and Design Tools. The sidebar menu allows users to access features quickly and efficiently.

## Layout

The interface is organized around a large workspace where users can create and edit designs. Tools and options are positioned logically, ensuring that users can focus on their design tasks without confusion.

## Typography

Canva uses clean and professional typography throughout the application. Font sizes, styles, and spacing are carefully designed to improve readability and create a visually appealing interface.

## Colors

The application uses bright and attractive colors while maintaining good contrast. The color scheme helps users distinguish between different tools and actions, improving usability.

## User Experience

Canva provides an excellent user experience through its drag-and-drop functionality, ready-made templates, and beginner-friendly design tools. Users can create professional designs with minimal effort, making the platform accessible to both beginners and experienced designers.
