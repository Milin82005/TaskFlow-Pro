# Activity 4 – Design Inspiration Research

## Design 1 – Dribbble

### Platform

Dribbble

### Observations

* The interface follows a clean and organized dashboard layout.
* Information is grouped into cards, making content easy to scan and understand.
* The navigation menu is clearly visible and accessible.
* Progress indicators and visual elements help users quickly understand important information.
* The design uses whitespace effectively to avoid clutter.

### Why It Inspired Me

* The layout is simple yet highly functional.
* Important information is highlighted without overwhelming the user.
* The design demonstrates excellent visual hierarchy.
* The interface provides a smooth and intuitive user experience.


## Design 2 – Behance

### Platform

Behance

### Observations

* The design maintains consistency across all sections.
* Typography is clear and easy to read.
* Soft colors create a modern and pleasant visual appearance.
* Components are arranged logically, helping users find information quickly.
* Task and activity sections are structured in a user-friendly manner.

### Why It Inspired Me

* The interface combines aesthetics with usability.
* Proper spacing and alignment improve readability.
* The design presents complex information in a simple way.
* It demonstrates effective use of modern dashboard design principles.


## Design 3 – Mobbin

### Platform

Mobbin

### Observations
